La escena · Miller §3

"Pinocho no mintió — y ese fue el problema"

🎬 Shrek Tercero · 2007
Príncipe Encantador: "Dime, marioneta, ¿dónde está Shrek?"

[Silencio. La nariz de Pinocho no crece]

Pinocho no mintió. No dijo nada que pudiera ser verdadero ni falso. Usó el lenguaje como cortina de humo. La misma estrategia aparece en publicidad de "superalimentos", discursos políticos y titulares diseñados para sonar sin comprometerse con ninguna afirmación verificable.

La lógica matemática estudia esto: ¿cuándo una afirmación puede ser verdadera o falsa? ¿Cuándo un argumento es sólido y cuándo tiene trampa?

Definición central · Miller §3.1

¿Qué es una proposición?

Una proposición es una oración declarativa que puede ser verdadera o falsa, pero no ambas. Las representamos con letras: p, q, r.

✓ Proposiciones:
"El ICBF atendió 23.000 casos en 2025." (V o F)
"La Luna orbita la Tierra." (verdadera)
"2+2=5." (falsa, pero proposición)
✗ No proposiciones:
"¿Cómo te fue en el parcial?" (pregunta)
"Estudia más." (mandato)
"¡Qué buena esa clase!" (exclamación)
Un enunciado compuesto une dos o más proposiciones con un conector lógico: y, o, no, si…entonces. Ojo: "La UdeC tiene sede en Facatativá y en Girardot" usa "y" para enumerar, no para conectar proposiciones. Es simple, no compuesto. (Miller: mismo caso con Smith & Wesson.)
Cuantificadores · Miller §3.1

¿Cuántos casos existen?

Universales: todos, cada, ninguno
"Todos los trastornos de personalidad afectan el funcionamiento social."
Existenciales: algunos, existe, al menos uno
"Algunos universitarios presentan ansiedad severa en parciales."
⚠️ Cómo negar correctamente — Miller Tabla 4
EnunciadoNegación correcta
Todos los X hacen YAlgún X NO hace Y
Algún X hace YNingún X hace Y
Ningún X hace YAlgún X hace Y

¿Por qué? Para refutar "todos los X hacen Y" solo necesito un contraejemplo. Por eso la negación es "algún X no hace Y" — no "ningún X hace Y", que sería una afirmación mucho más fuerte.

Practica:

Conexión con conjuntos: "Todos los A son B" ≡ A ⊂ B. "Ningún A es B" ≡ A ∩ B = ∅. "Algún A es B" ≡ A ∩ B ≠ ∅.

Cómo combinar proposiciones

Explorador interactivo — toca p o q

Tablas de verdad en tiempo real

pV

Expresión con 3 variables — paso a paso

¿Qué pasa cuando mezclamos conectivos?

Definición de las variables (V = esa proposición ES verdadera · F = ES falsa)
p = "El estudiante está inscrito en prácticas" → V (sí lo está)
q = "El supervisor firmó el aval" → F (no lo firmó aún)
r = "El horario está confirmado" → V (sí está confirmado)
Evalúa: ¬p ∧ (q ∨ ¬r)
Paso 1: ¬p = ¬(V) = F — "El estudiante NO está inscrito" es FALSO
Paso 2: ¬r = ¬(V) = F — "El horario NO está confirmado" es FALSO
Paso 3: q ∨ ¬r = F ∨ F = F — ni el aval ni el horario
Paso 4: ¬p ∧ (q ∨ ¬r) = F ∧ F = F
Orden: primero paréntesis, luego negaciones, luego ∧ y ∨.
Leyes de De Morgan · Miller §3.2

Cómo negar una proposición compuesta

De Morgan 1
¬(p∧q) = ¬p∨¬q
Para que la asistencia cuente, debes llegar temprano Y entregar el taller. Si falla uno, no cuenta.

p = “Llegüé temprano” (V si sí, F si no)
q = “Entregué el taller” (V si sí, F si no)

¬(p∧q) = “No es cierto que llegüé y entregué”
= ¬p∨¬q = “No llegüé O no entregué”
El ‘y’ pasa a ‘o’. Basta que falle UNO.
Zona teal = p∧q: llegaron temprano y entregaron el taller.
✓ ¡Misma zona! ¬(p∧q) y ¬p∨¬q son el mismo conjunto. Eso es De Morgan 1.
De Morgan 2
¬(p∨q) = ¬p∧¬q
Para pasar basta aprobar el primer corte O el segundo. Para no pasar, fallan los dos.

p = “Aprobé el primer corte” (V si sí, F si no)
q = “Aprobé el segundo corte” (V si sí, F si no)

¬(p∨q) = “No es cierto que aprobé el primero o el segundo”
= ¬p∧¬q = “No aprobé el primero Y no aprobé el segundo”
El ‘o’ pasa a ‘y’. Tienen que fallar LOS DOS.
Zona teal = p∨q: aprobaron el primer corte o el segundo (o los dos).
✓ ¡Misma zona! ¬(p∨q) y ¬p∧¬q son el mismo conjunto. Eso es De Morgan 2.
Regla: Al negar, el conector se INVIERTE (∧↔∨) y cada parte SE NIEGA. Siempre las dos cosas juntas.

El conector de las hipótesis · Miller §3.3

El condicional: "Si p… entonces q"

El condicional p → q es la estructura formal de las hipótesis científicas. p es el antecedente (la condición); q es el consecuente (lo que se garantiza si la condición ocurre).

La analogía de la promesa

Tus papás te prometen: "Si apruebas todas las materias este semestre, te pagamos el viaje a San Andrés."

p = "apruebas todas las materias" — puede ser V (sí aprobaste) o F (no aprobaste)
q = "tus papás te pagan el viaje a San Andrés" — puede ser V (sí pagan) o F (no pagan)

¿Cuándo habrían mentido tus papás? Toca cada caso:

La regla: p → q es FALSO únicamente cuando p=V y q=F. Solo cuando la condición se cumplió y la promesa no se honró.
pqp→qSignificado
VVVAprobaste y te pagaron. Promesa cumplida. ✓
VFFAprobaste y NO te pagaron. ¡Promesa rota! ✗
FVVNo aprobaste pero te pagaron igual. Generosos, no mintieron. ✓
FFVNo aprobaste y no te pagaron. La condición no se activó. ✓
Equivalencia: p→q es lo mismo que ¬p∨q.
